# Negeri Sembilan 2026 State Election Prediction

## Overview
This project aims to predict the election outcomes for the 36 state assembly (DUN) seats in Negeri Sembilan for the 2026 state election. By utilizing historical election data from 2013, 2018, and 2023, this model provides a data-driven forecast of seat winners.

## Background
Following the dissolution of the Negeri Sembilan State Legislative Assembly in June 2026, the state is heading to the polls on August 1, 2026. This project explores the political landscape as Pakatan Harapan and other coalitions contest for the 36 available seats.

## Methodology
The prediction model employs three machine learning algorithms to classify the likely winning party for each constituency:
- **Logistic Regression**
- **Random Forest Classifier**
- **Support Vector Machines (SVM)**

The current scope focuses exclusively on predicting the seat winner, rather than coalition formations or Chief Minister appointments.
